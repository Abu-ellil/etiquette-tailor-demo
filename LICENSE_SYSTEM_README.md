# نظام حماية التطبيق بالـ Hardware ID
## Etiquette Tailor - License Protection v2.0

تم تطبيق نظام حماية **مرتبط بجهاز العميل** - المفتاح يعمل فقط على جهاز واحد!

---

## 🔐 كيف يعمل Hardware ID؟

1. **العميل يفتح التطبيق** → يظهر له **Hardware ID** خاص بجهازه
2. **العميل يرسل لك الـ Hardware ID**
3. **أنت تولد مفتاح مرتبط بذلك الـ Hardware ID**
4. **العميل يدخل المفتاح** → يعمل فقط على نفس الجهاز

---

## 📋 سير العمل الكامل

### الخطوة 1: العميل يحصل على Hardware ID

عند فتح التطبيق لأول مرة، يظهر للعميل:

```
┌─────────────────────────────────────┐
│   Your Hardware ID                  │
│   ┌─────────────────────────────┐   │
│   │ HWID-A1B2C3D4E5F67890       │   │
│   └─────────────────────────────┘   │
│   [Copy]                            │
└─────────────────────────────────────┘
```

### الخطوة 2: العميل يرسل لك رسالة:

```
مرحباً،

أريد شراء ترخيص لـ Etiquette Tailor.

بياناتي:
- الاسم: شركة الأمل للخياطة
- البريد: info@al-amal.com
- Hardware ID: HWID-A1B2C3D4E5F67890

إلى اللقاء.
```

### الخطوة 3: أنت تولد المفتاح:

```bash
npm run generate-license -- \
  --hwid "HWID-A1B2C3D4E5F67890" \
  --name "شركة الأمل للخياطة" \
  --email "info@al-amal.com" \
  --type full
```

### الخطوة 4: ترسل للعميل:

```
مرحباً،

إليك مفتاح ترخيصك:
ETIK-A1B2-C3D4-E5F6-7890

⚠️ مهم: هذا المفتاح مرتبط بجهازك فقط (Hardware ID: HWID-A1B2...)
لن يعمل على أي جهاز آخر.

افتح التطبيق → أدخل المفتاح → اضغط تفعيل
```

### الخطوة 5: العميل يفعل التطبيق:

- يدخل المفتاح
- التطبيق يتحقق أن الـ Hardware ID يطابق
- التفعيل بنجاح! ✅

## 🖥️ ما معلومات الـ Hardware ID؟

الـ Hardware ID يتولد من:

| المعلومة | الغرض |
|----------|-------|
| Machine GUID / UUID | معرف فريد للجهاز |
| عنوان MAC الأول | بطاقة الشبكة الأساسية |
| معلومات CPU | نوع المعالج |
| حجم الذاكرة RAM | معلومات إضافية |
| اسم الجهاز | تأكيد إضافي |

كل هذه المعلومات تُجمع وتُشفّر لتنتج معرف فريد مثل:
```
HWID-A1B2C3D4E5F67890
```

## 🚨 حالات الخطأ

### 1. Hardware ID Mismatch
إذا حاول العميل تشغيل التطبيق على جهاز آخر:

```
┌──────────────────────────────────────────────┐
│  ⚠️ License Locked to Another Device         │
│                                              │
│  Your license is activated on a different    │
│  computer. Please contact support to         │
│  transfer it.                                │
└──────────────────────────────────────────────┘
```

### 2. الحلول للعميل:

**خيار 1: شراء ترخيص إضافي**
- لكل جهاز يحتاج ترخيص منفصل

**خيار 2: نقل الترخيص** (يمكنك إضافته مستقبلاً)
- إلغاء التفعيل من الجهاز القديم
- تفعيل على الجهاز الجديد

## 🔒 الأمان

### المستوى الحالي:
- ✅ المفتاح مرتبط بجهاز واحد
- ✅ لا يمكن نسخ الترخيص لأجهزة أخرى
- ✅ التحقق محلي (بدون إنترنت)
- ✅ معلومات الجهاز لا تُرسل لأي سيرفر

- تقييد عدد مرات التفعيل
- ربط بحساب عميل عبر الإنترنت
